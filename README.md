# CFAI Project - Algorithm Complexity Comparator

This project is a web application that compares algorithmic time complexities.
It allows users to generate datasets and compare the execution time of:
1. Linear Search vs Binary Search
2. Subset Checking (Lists vs Sets)

## Team Work
This repository tracks the contributions of our 2-person team.

## Getting Started
1. Clone the repository.
2. Run `python -m venv venv` to create a virtual environment.
3. Activate the environment and `pip install -r requirements.txt`.
4. Run `python app.py`.
